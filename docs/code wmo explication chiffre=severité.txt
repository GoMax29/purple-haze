1) “Un code plus élevé = plus sévère” : vrai… mais seulement à l’intérieur d’une même famille
Dans la table WMO 4677 (temps présent “ww”), les familles 51/53/55 (bruine), 61/63/65 (pluie), 71/73/75 (neige) et 80/81/82 (averses) codent faible → modéré → fort/violent quand le nombre augmente. Idem pour 56/57 (bruine verglaçante légère → modérée/forte), 66/67 (pluie verglaçante légère → modérée/forte) et 85/86 (averses de neige légère → modérée/forte). 
nodc.noaa.gov

En revanche, on ne peut pas comparer la sévérité entre familles en se basant juste sur le numéro. Exemple : 66 (pluie verglaçante légère) est beaucoup plus dangereux que 61 (pluie faible) malgré un numéro proche. 
nodc.noaa.gov

Important : les termes faible/modéré/fort ne sont pas normalisés mondialement en mm/h (pluie) ou en visibilité (neige) par le WMO ; différents services nationaux emploient des seuils différents. Autrement dit, l’étiquette d’intensité est qualitative. 
avsef.gov.au

⚠️ Sur 0–3 : dans la table WMO d’origine, 00–03 décrivent l’évolution de la nébulosité (nuages en formation/dissipation…), pas “ciel clair / peu nuageux / couvert”. Ce dernier triplet vient d’une simplification d’API (Open-Meteo & co), pas du WMO brut. 
nodc.noaa.gov

Notes d’interprétation :
• Dans 4677, les numéros pairs/impairs distinguent parfois “intermittent/continu” (ex. 50/51, 60/61…). Tu utilises les continus (51/53/55, 61/63/65, 71/73/75), ce qui simplifie l’UX. 
nodc.noaa.gov

• Les “averses” (80–86) sont convectives (courtes mais intenses) — d’où 82 “violent”, même si la lame d’eau totale peut rester modeste. 
nodc.noaa.gov

4) Faut-il une icône par code ?
Pas forcément. Le retour d’expérience (Met Office, DWD, communautés FR/DE) montre qu’une base d’icônes par phénomène + modificateurs transmet mieux l’info que 28 pictos tous différents. 
Met Office
Deutscher Wetterdienst
Les Forums d'Infoclimat

Recommandation pragmatique :

Familles d’icônes : ciel clair / nuageux / brouillard / bruine / pluie / neige / averses / orage.

Modificateurs : intensité (faible/modéré/fort), “verglaçant” (badge flocon+goutte), “grêle” (badge grêlon), “jour/nuit”.

Séparation utile : “continu” vs “averses” (pluie convective) — c’est bien perçu par l’utilisateur.

Option : un picto spécial “brouillard givr.” (48) et “neige grains” (77) si tu vises la précision routière.

Tu arrives alors à ~15–18 variantes lisibles, tout en gardant une correspondance 1→n avec tes 28 codes (même base + modif.). Si tu veux tout de même un set exhaustif ww→icône, il existe des packs “4677” open-source pour s’inspirer. 
GitHub
